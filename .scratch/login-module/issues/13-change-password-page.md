# 修改密码页面

## Parent

参考 `.scratch/login-module/PRD.md`

## What to build

实现修改密码页面UI组件，支持首次登录强制修改、密码过期修改、普通修改三种场景。

核心功能：
- 原因提示（首次登录/密码过期/普通修改）
- 旧密码输入框（非首次登录时显示）
- 新密码输入框
- 密码强度指示器（进度条+文字：弱/中等/强）
- 密码要求检查列表（8位、字母、数字三项实时检查）
- 确认密码输入框
- 提交按钮
