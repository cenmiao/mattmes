   | [Lock图标] 密码输入框   |

3. **登录API封装** (`src/api/auth.ts`):
   ```typescript
   interface LoginRequest {
     userNo: string
     password: string
     forceLogin?: boolean
   }
   
   interface LoginResponse {
     token: string
     userInfo: {
       id: number
       userNo: string
       name: string
       roles: Array<{ roleCode: string, roleName: string }>
       permissions: string[]
     }
     needChangePassword: boolean
   }
   
   function login(data: LoginRequest): Promise<LoginResponse>
   ```

4. **登录成功处理**:

### 密码锁定时间格式化

后端返回毫秒数，前端格式化为"X分X秒":
```typescript
function formatLockTime(ms: number): string {
  const minutes = Math.floor(ms / 60000)
  const seconds = Math.floor((ms % 60000) / 1000)
  return `${minutes}分${seconds}秒`
}
```
