### 关键实现点

1. **Layout组件结构**:
   ```
   +------------------+------------------------+
   |     Sidebar      |        Header          |
   |     (240px)      |        (64px)          |
   |                  +------------------------+
   |     菜单列表      |        Content         |
   |     (可折叠)      |    <router-view />     |
   +------------------+------------------------+
   ```

2. **动态菜单逻辑**:
   - 从 `localStorage.getItem('userInfo')` 获取用户权限列表
   - 遍历 `menus.ts` 配置，过滤掉用户无权限的菜单项
   - 超级管理员（角色编码 SUPER_ADMIN）显示所有菜单

3. **用户下拉菜单**:
   - 显示用户名和工号（从 userInfo 获取）
   - "修改密码" — 跳转 `/change-password`
   - "退出登录" — 清空 localStorage、调用后端接口、跳转 `/login`

4. **路由结构**:
   ```typescript
   /                    → 重定向到 /dashboard
   /login               → Login.vue (public)
   /403                 → 403.vue (public)
   /404                 → 404.vue (public)
   /change-password     → ChangePassword.vue (需登录)
   /                    → Layout.vue (嵌套路由容器)
     /dashboard         → Dashboard.vue
     /users             → UserList.vue (permission: user:view)
     /roles             → RoleList.vue (permission: role:view)
     /permissions       → PermissionList.vue (permission: permission:view)
     /login-logs        → LoginLogList.vue (permission: login-log:view)
   ```

5. **退出登录API**:
   - 调用 `POST /api/logout` 更新登录日志 `logout_type=MANUAL`
   - 如果后端接口不存在（#09 未完成），可暂时跳过API调用，仅清空前端状态

### 超级管理员判断

```typescript
const userInfo = JSON.parse(localStorage.getItem('userInfo') || '{}')
const isSuperAdmin = userInfo.roles?.some((r: any) => r.roleCode === 'SUPER_ADMIN')
```

超级管理员跳过权限检查，显示所有菜单。
