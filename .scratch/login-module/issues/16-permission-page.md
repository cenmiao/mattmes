**TypeScript类型定义**:
```typescript
export interface PermissionResponse {
  id: number
  permissionName: string
  permissionCode: string
  permissionType: number  // 1=模块, 2=按钮
  parentId: number | null
  description: string | null
  createTime: string
}

export interface PermissionTreeResponse {
  id: number
  permissionName: string
  permissionCode: string
  children: PermissionResponse[]
}

export interface PermissionCreateRequest {
  permissionName: string
  permissionCode: string
  parentId?: number  // 有值表示按钮级权限
  description?: string
}

export interface PermissionUpdateRequest {
  permissionName: string
  description?: string
}
```

### 关键实现点

1. **树形展开表格**:
   - 使用Element Plus Table的`row-key="id"`和默认展开功能
   - 模块级权限（permissionType=1）可展开显示按钮级子权限
   - 按钮级权限（permissionType=2）无子节点

2. **表格列设计**:
   - 权限名称：模块级显示`<Folder />`图标，按钮级显示`<Pointer />`图标
   - 权限编码：等宽字体(`--font-data`)，Cyan色(`#22D3EE`)
   - 类型列：Badge徽章 — 模块(蓝色`type="primary"`)、按钮(灰色`type="info"`)
   - 描述列：普通字体，超长省略
   - 创建时间：等宽字体
   - 操作列：图标按钮组（编辑、删除、新增子权限）

3. **新增权限弹窗**:
   - 权限名称输入（必填）
   - 权限编码输入（必填，placeholder提示格式如`user`或`user:add`）
   - 父权限选择（可选下拉框，选择后自动变为按钮级权限）
   - 描述输入（可选文本域）
   - 提交按钮

4. **编辑权限弹窗**:
   - 权限名称、描述可编辑
   - 权限编码只读显示（不可修改）
   - 提交按钮

5. **新增子权限按钮**:
   - 仅在模块级权限行显示
   - 点击打开新增弹窗，自动填充父权限下拉框

6. **删除权限**:
   - 确认弹窗提示
   - 删除模块级时额外提示"将同时删除其下所有按钮级权限"
   - 删除成功刷新列表

7. **权限控制**:
   ```vue
   <el-button v-permission="'permission:add'" ...>新增</el-button>
   <el-button v-permission="'permission:edit'" ...>编辑</el-button>
   <el-button v-permission="'permission:delete'" ...>删除</el-button>
   ```
