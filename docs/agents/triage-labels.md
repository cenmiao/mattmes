# Triage Labels

技能使用五个规范的分流角色。此文件将这些角色映射到本仓库问题追踪器中实际使用的标签字符串。

| mattpocock/skills 中的角色 | 本仓库追踪器中的标签 | 含义 |
| -------------------------- | -------------------- | ---- |
| `needs-triage` | `needs-triage` | 需要维护者评估此问题 |
| `needs-info` | `needs-info` | 等待报告者补充更多信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整,可由 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不予处理 |

当技能提及某个角色(如"应用 AFK-ready 分流标签"),使用此表中对应的标签字符串。

可编辑右侧列以匹配你实际使用的词汇。
